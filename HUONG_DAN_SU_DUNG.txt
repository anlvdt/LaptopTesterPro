🔧 YÊU CẦU HỆ THỐNG:
   • Windows 10/11 (khuyến nghị)
   • Python 3.8 trở lên
   • RAM: 4GB+
   • Dung lượng trống: 500MB+
   • Quyền Administrator (cho một số tính năng)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 CÁCH 1: CHẠY NHANH (KHUYẾN NGHỊ)

   1. Giải nén file LaptopTester_Backup.zip
   2. Mở thư mục LaptopTester
   3. Nhấp đúp vào file: run.bat
   4. Chờ cài đặt tự động (lần đầu tiên)
   5. Ứng dụng sẽ tự động khởi động!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔨 CÁCH 2: CÀI ĐẶT THỦ CÔNG

   1. Mở Command Prompt (cmd) hoặc PowerShell
   2. Di chuyển đến thư mục LaptopTester:
      cd C:\MyApps\LaptopTester
   
   3. Cài đặt thư viện:
      pip install -r requirements.txt
   
   4. Chạy ứng dụng:
      python main_enhanced_auto.py

📋 CÁC CHỂ ĐỘ SỬ DỤNG:

   🟢 BASIC MODE (Người dùng thông thường):
      • 13 bước kiểm tra cơ bản
      • Phù hợp cho người mua laptop cũ
      • Thời gian: 15-20 phút
      • Bao gồm: Hardware, License, Screen, Keyboard, Battery, Audio, Webcam...

   🔴 EXPERT MODE (Chuyên gia):
      • 17+ bước kiểm tra nâng cao
      • Bao gồm tất cả test của Basic Mode
      • Thêm: CPU Stress Test, GPU Stress Test, Thermal Monitor...
      • Thời gian: 30-45 phút
      • Phù hợp cho kiểm tra chuyên sâu

🐛 XỬ LÝ SỰ CỐ:

   ❌ Lỗi "Python not found":
      → Cài đặt Python từ: https://www.python.org/downloads/
      → Chọn "Add Python to PATH" khi cài đặt

   ❌ Lỗi "pip not found":
      → Chạy: python -m ensurepip --upgrade

   ❌ Lỗi "Permission denied":
      → Chạy cmd/PowerShell với quyền Administrator
      → Chuột phải → "Run as administrator"

   ❌ Lỗi "Module not found":
      → Chạy lại: pip install -r requirements.txt
      → Hoặc: pip install customtkinter psutil pillow opencv-python

   ❌ Ứng dụng không hiển thị:
      → Kiểm tra có thông báo lỗi trong cmd không
      → Thử chạy: python main_enhanced_auto.py
      → Xem file logs/error.log
