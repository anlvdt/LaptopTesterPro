Q: File EXE có an toàn không?
A: Có! Windows Defender cảnh báo vì chưa có chữ ký số. Chọn "Run anyway".

Q: Cần kết nối Internet không?
A: Không! Chạy offline hoàn toàn.

Q: Có thể chạy trên USB không?
A: Có! Copy thư mục LaptopTester_Portable/ vào USB là xong.

Q: Làm sao build lại file EXE?
A: Click đúp BUILD_EXE.bat (cần Python đã cài)

Q: File EXE quá lớn (89MB)?
A: Bình thường! Đã bao gồm tất cả thư viện cần thiết.
